Project Description:

The 3D Car Driving Game is an interactive computer graphics project developed using C++, OpenGL, and GLUT. The objective of the game is to control a vehicle on a 3D road environment while avoiding obstacles, collecting coins, and achieving the highest possible score. The project demonstrates several fundamental and advanced concepts of computer graphics, including 3D object modeling, texture mapping, lighting, camera control, animation, collision detection, and real-time user interaction. The game environment contains various graphical elements such as roads, trees, houses, mountains, billboards, street lamps, and a windmill, creating a visually engaging driving experience. The player controls a car that can move between lanes to avoid incoming obstacles and collect coins placed on the road. A scoring system tracks the player's performance, while a lives system determines game progression. Dynamic lighting and texture mapping are used to improve realism and visual quality. Multiple camera views and keyboard controls allow the user to interact with the environment from different perspectives. The project integrates graphics rendering, gameplay mechanics, and user interface elements into a complete and functional 3D game. It serves as a practical application of computer graphics concepts learned throughout the course and demonstrates teamwork in software development, game design, and graphical programming.

Features:

1. 3D vehicle rendering
2. Real-time obstacle generation
3. Coin collection system
4. Score and lives management
5. Collision detection
6. Dynamic lighting effects
7. Texture mapping
8. Multiple camera views
9. Interactive keyboard controls
10. Environment modeling with trees, houses, mountains, billboards, and windmill
11. Heads-Up Display (HUD) for game information
